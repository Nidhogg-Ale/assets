# koishi-plugin-assets-s3
## 配置项

### region

- 类型: `string`
- example us-west-rack-2
服务器所属地域。

### endpoint

- 类型: `string`
- example https://s3.example.amazonaws.com
- example http://localaddress:9001

远程服务器地址。

### accseeKeyId

- 类型: `string`
- example KFCVIVO50CRAZYThursd

用户keyid,在对象存储控制台生成

### secretaccseeKey

- 类型: `string`
- example KFCVIVO50CRAZYThursday1145141919810SWOv

用户私钥一般为40位，与上一个同时生成。

### bucket

- 类型: `string`

存储桶名称。

### pathPrefix

- 类型: `string`

路径前缀。

### publicUrl

- 类型: `string`
